About the Data

  • The dataset includes Covid-19 related Tweets from January 27, 2020 to September 30, 2020.
  • The dataset was collected using the Twitter API
  • The total number of tweets in the dataset is 91448. All metadata provided by the Twitter API were persisted.

Feature Engineering

  • We extracted the sentiment scores and sentiment labels for each tweet using Transformers.
  • We extracted significant phrases or topics from each unstructured Tweet text in an attempt to organize, understand and summarize large collections of highly unstructured textual information stored in the covid-19 twitter text. We used the rake-nltk library for topic modeling
  • Sentiment metadata and topic metadata were appended to the original dataset
